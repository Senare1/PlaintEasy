Questions Fréquemment Posées (FAQ)

Pour créer un compte, renseigner vos informations dans le formulairede creation ' l'acceui puis valider avec le bouton S'inscrire en bas du formulaire.C'est aussi simple, Vous devrez entrer votre adresse e-mail, choisir un mot de passe, et confirmer votre inscription via un e-mail de vérification.

Pour réinitialiser votre mot de passe, cliquez sur "Mot de passe oublié ?" sur la page de connexion. Suivez les instructions pour recevoir un e-mail avec un lien de réinitialisation.

Vous pouvez contacter notre support technique en utilisant le formulaire de contact disponible dans la section "Contactez-nous" de notre site en bas de cette page.

Pour déposer une plainte, veuillez suivre ces étapes :

  1. Remplir le Formulaire : Accédez au formulaire de dépôt de plainte sur notre plateforme.
  2. Décrire le Problème : Détaillez la situation ou le problème rencontré en fournissant toutes les informations nécessaires.
  3. Soumettre : Envoyez le formulaire. Vous recevrez une confirmation de réception par e-mail.
  4. Suivi : Notre équipe examinera votre plainte et vous contactera si nécessaire pour des informations complémentaires.

Nous nous engageons à traiter toutes les plaintes avec sérieux et à vous fournir une réponse rapide.
